NDS DB API

Documentation de l'API REST pour la base de données de jeux Nintendo DS.

Base URL: /api

Endpoints

GET /api/games

Récupère la liste principale et minimaliste des jeux. Idéal pour les recherches rapides et l'indexation.

Exemples d'utilisation

/api/games

Exemple de Réponse (minimaliste)

[
  {
    "id": "A2DE",
    "serial": "NTR-A2DE-USA",
    "title": "Animal Crossing: Wild World"
  },
  {
    "id": "YQUE",
    "serial": "NTR-YQUE-EUR",
    "title": "Professor Layton and the Curious Village"
  }
  ...
]
GET /api/games/:id

Récupère les données complètes et riches d'un jeu spécifique en utilisant son id (ex: A2DE) ou son serial (ex: NTR-A2DE-USA).

Cette route fusionne les données de base avec le meta.json correspondant et construit dynamiquement les URLs des assets.

Exemples d'utilisation

/api/games/A2DE

GET /api/search

Recherche des jeux dont le titre correspond à la requête (basé sur la liste minimaliste).

Paramètres de Requête (Query)

  • title (requis): Le terme de recherche à inclure dans le titre.

Exemples d'utilisation

/api/search?title=layton

Exemple de Réponse (minimaliste)

[
  {
    "id": "YQUE",
    "serial": "NTR-YQUE-EUR",
    "title": "Professor Layton and the Curious Village"
  }
]
