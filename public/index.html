NDS DB API

Documentation de l'API REST pour la base de données de jeux Nintendo DS.

Base URL: /api

Endpoints

GET /api/games

Récupère la liste complète des jeux NDS. Accepte des filtres en paramètres de requête.

Paramètres de Requête (Query)

  • region (optionnel): Filtre les jeux par région (ex: USA, EUR, JPN).
  • publisher (optionnel): Filtre les jeux par éditeur (sensible à la casse, recherche partielle).

Exemples d'utilisation

Pour tous les jeux :

/api/games

Pour les jeux de la région USA :

/api/games?region=USA

Pour les jeux publiés par "Nintendo" :

/api/games?publisher=Nintendo

Exemple de Réponse (raccourci)

[
  {
    "id": "A2DE",
    "serial": "NTR-A2DE-USA",
    "title": "Animal Crossing: Wild World",
    "region": "USA",
    ...
  },
  ...
]
GET /api/games/:id

Récupère un jeu spécifique en utilisant son id (ex: A2DE) ou son serial (ex: NTR-A2DE-USA).

Paramètres d'URL

  • :id (requis): L'identifiant unique ou le numéro de série du jeu.

Exemples d'utilisation

/api/games/A2DE
/api/games/NTR-YQUE-EUR

Exemple de Réponse

{
  "id": "A2DE",
  "serial": "NTR-A2DE-USA",
  "title": "Animal Crossing: Wild World",
  "region": "USA",
  "publisher": "Nintendo",
  "release_date": "2005-12-05",
  "languages": ["en", "fr", "es"]
}
GET /api/search

Recherche des jeux dont le titre correspond à la requête. La recherche n'est pas sensible à la casse.

Paramètres de Requête (Query)

  • title (requis): Le terme de recherche à inclure dans le titre.

Exemples d'utilisation

/api/search?title=layton
/api/search?title=animal

Exemple de Réponse

[
  {
    "id": "YQUE",
    "serial": "NTR-YQUE-EUR",
    "title": "Professor Layton and the Curious Village",
    "region": "EUR",
    ...
  }
]
